# GearX

A web-based dashboard for **CarX Street** tuning purposes based on [StepUnique's calculator](https://www.youtube.com/watch?v=P2YeIHZyy8c).

Please don't take this as an absolute recommendation, this is just a tool to calculate your reference values for your car when tuning them.
**I made this for fun**, for my personal use, and it's definitely **NOT** backed by actual accurate physics, it's just a game, so take everything with a grain of salt :)

Most of the formula are based on [StepUnique's spreadsheet](https://docs.google.com/spreadsheets/d/1wCFODspC2uiHsKwhUp1FUeWh5b8ys4YGXb46TU-qm-o/edit?gid=934152913#gid=934152913). Credit goes to them for making it.
A lot of the things here are derived based on that, with some tweaks and additions I made. I am not an engineer, so I don't know if this is correct or not, but it works okay in the game so eh :p

Everything on this app should be self-explanatory, if you need some hint press the `(?)` button and it should give you some relevant information you need.

If you have any suggestions, feedback, or anything, feel free to [visit the Github repository](https://github.com/elianiva/gearx) or [open an issue](https://github.com/elianiva/gearx/issues) :)
